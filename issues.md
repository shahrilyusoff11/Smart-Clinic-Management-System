# User Stories for Smart Clinic Management System

## Doctor
1. As a Doctor, I want to view a list of all my booked patient appointments so that I can prepare for my day.
2. As a Doctor, I want to manage my available times so that patients can only book appointments when I am free.
3. As a Doctor, I want to write prescriptions for specific appointments so that my patients receive the correct medical treatment.

## Patient
1. As a Patient, I want to search and find a doctor by name so that I can easily book an appointment with them.
2. As a Patient, I want to book an appointment with a doctor at an available time so that I can get medical assistance.
3. As a Patient, I want to view my past prescriptions so that I can keep track of my medication history.

## Admin
1. As an Admin, I want to add a new Doctor to the system so that new staff can start taking appointments.
2. As an Admin, I want to view daily appointment reports for any doctor so that I can monitor clinic activity.
3. As an Admin, I want to remove or deactivate user accounts so that the system remains secure and up-to-date.
